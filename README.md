# Мобильное приложение ServiceBook 

***Репозиторий с Frontend частью приложения ServiceBook***

## Стек технологий
| Компонент | Технология                       |
| --------- | -------------------------------- |
| Frontend  | React Native + Expo + TypeScript |
| Backend   | Node.js + Express + TypeScript   |
| БД        | PostgreSQL                       |
| Auth      | JWT (access + refresh)           |
| Push      | Expo Notifications               |

 ## Что может приложение?
|**Возможности приложения**  | **Статус** |
| --- | --- |
| Интуитивно понятный интерфейс  | ✔  |
|  Авторизация по e-mail | ✔  |
| Гараж для каждого аккаунта  | ✔  |
| Напоминания (дата / пробег) для деталей автомобилей  | ✔  |
| Push-уведомления (Expo)  | ✔  |
| Фильтрация по тегам  | ✔  |

### API Endpoints

| Метод    | Путь                  | Описание             |
| -------- | --------------------- | -------------------- |
| `GET`    | `/cars`               | Список автомобилей   |
| `POST`   | `/cars`               | Добавить авто        |
| `GET`    | `/details/:carId`     | Напоминания авто     |
| `POST`   | `/details/:carId`     | Создать напоминание  |
| `PATCH`  | `/details/:carId/:id` | Обновить напоминание |
| `DELETE` | `/details/:carId/:id` | Удалить напоминание  |
| `GET`    | `/details/tags`       | Все теги             |
| `POST`   | `/user/refresh`       | Обновить токены      |
